## Httpmessageconverter

-   JSON 데이터를 HTTP 메시지 바디 내 직접 읽거나 쓰는 경우에 사용
-   String 처리는 StringHttpMessageConverter 사용
-   Object 처리는 MappingJackson2HttpMessageConverter 사용
